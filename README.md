ავტომობილების გაყიდვების დეშბორდი — Power BI პროექტი
📌 პროექტის მიმოხილვა

ამ პროექტის მიზანია დინამიკური და ინტერაქტიული ავტომობილების გაყიდვების დეშბორდის შექმნა Power BI-ში, რომელიც საშუალებას იძლევა რეალურ დროში შეფასდეს გაყიდვების შედეგები და ტენდენციები.
